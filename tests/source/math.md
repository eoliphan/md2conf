## LaTeX math equations

Inline formulas can be delimited with `\(` and `\)`: \(\sum_{i=1}^{n} i = \frac{n(n+1)}{2}\), or another example: \(\lim _{x\rightarrow \infty }\frac{1}{x}=0\).

Block formulas can be wrapped with `\[` and `\]`:

\[\frac{-b\pm \sqrt{b^{2}-4ac}}{2a}\]

Code blocks with language `math` are also supported:

```math
\int _{a}^{b}f(x)dx=F(b)-F(a)
```
